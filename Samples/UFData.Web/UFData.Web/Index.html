Data format

The schedule data is in JSON format (application/json)

  • [...] array of schedule item
  • [Schedule item:
    • Teachers: string array
    • StartTimeUtc: start time defined in UTC time in ISO 8601 string format
    • Duration: a timespan (dd:HH:mm)
    • Class: the group identity as string
    • Room: the room identity as string
    • Title: subject as string
